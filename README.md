# China Antiga – Projeto Arduino

Este repositório contém o código em **Arduino (C++)** desenvolvido para o projeto **"China Antiga"**, apresentado no **Rá-Tim-Bum 2025** da **Universidade Santa Cecília (UNISANTA)**.

## Reconhecimento

O projeto foi **ganhador do segundo lugar**, competindo com projetos de **todas as engenharias** da universidade.

## Objetivo do Projeto

O objetivo do projeto foi representar elementos da **China Antiga** de forma interativa, aplicando conceitos de **programação embarcada, eletrônica e automação**, com foco educacional e cultural.

## Tecnologias Utilizadas

- Arduino
- Linguagem C++
- Fita RGB
- Módulo MP3 DFPlayer
- Alto-falantes
- LEDs
- Sensor LDR
